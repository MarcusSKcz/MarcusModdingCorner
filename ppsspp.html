PPSSPP

The best PSP emulator available — runs on everything from PC to Android.

What is PPSSPP

PPSSPP is a fast and portable PSP emulator that runs on nearly every platform including Windows, Linux, Android, iOS and Mac. It has excellent compatibility with the PSP library and offers enhancement features that make PSP games look significantly better than on original hardware.

Key Features

Resolution upscaling up to 10x native. Texture scaling and filtering. Anisotropic filtering support. Controller support including DualShock and Xbox controllers. Save state support. Works great on Android phones making it the go-to mobile emulator.

Recommended Specs

Extremely lightweight — any modern dual core CPU with 2GB of RAM handles it perfectly. On Android any phone from the last 5 years will run it at full speed. The emulator is well optimised for low-end hardware.

Recommended Settings

Set rendering resolution to 4x or higher. Enable anisotropic filtering at 16x. Use Vulkan backend on PC for best performance. On Android use Vulkan if your device supports it, otherwise OpenGL. Enable texture scaling with xBRZ for smoother textures.
