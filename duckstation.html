Duckstation

The most accurate and feature-rich PlayStation 1 emulator available.

What is Duckstation

Duckstation is a modern PlayStation 1 emulator focused on accuracy, performance and usability. It supports the full PS1 library with high compatibility and offers a wide range of enhancement features that make classic games look and feel better than ever.

Key Features

Upscaling up to 8x native resolution for drastically sharper visuals. Widescreen patches for games that originally ran in 4:3. PGXP geometry correction which eliminates the wobbly polygon effect that plagued PS1 games. Texture replacement support and analog precision correction.

Recommended Specs

Duckstation is extremely lightweight. Any modern dual core CPU running at 2.0GHz with 4GB of RAM will run it perfectly. Even low-end hardware handles it without issues.

Recommended Settings

Set the renderer to Vulkan for best performance. Enable PGXP Geometry Correction for cleaner visuals. Set internal resolution to 4x or higher if your hardware allows. Enable widescreen rendering for supported games.
