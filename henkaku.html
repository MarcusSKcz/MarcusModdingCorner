Henkaku Enso Setup

Install permanent CFW on your PS Vita on firmware 3.60 or 3.65.

⚠️ This guide is for educational purposes only. Proceed at your own risk.

What you need

A PS Vita on firmware 3.60 or 3.65

A Wi-Fi connection

A PSN account (optional but recommended)

Step 1 — Check your firmware

Go to Settings → System → System Information. You need firmware 3.60 or 3.65. If you are on a higher firmware unfortunately you cannot install Henkaku at this time.

Step 2 — Install Henkaku

On firmware 3.60, open the browser on your Vita and navigate to henkaku.xyz. Tap Install and follow the prompts. The exploit will run automatically and install the Henkaku homebrew enabler.

Step 3 — Install VitaShell

After Henkaku is installed, open the molecularShell app that appears on your home screen. This gives you access to the file system and allows you to install VPK files.

Step 4 — Install Enso

Download the Enso installer VPK and transfer it to your Vita via USB or FTP. Install it through VitaShell, then launch Enso from the home screen and follow the installation steps.

Step 5 — Done

Enso is now permanently installed. Your Vita will boot into CFW every time without needing to run an exploit. You can now install plugins, themes, and homebrew applications freely.
