# Marcus' Modding Corner

A web portal about console modding and emulation built for the Web Technologies & Design course.

## Pages

- **Home** - Main landing page
- **Guides** - Console modding guides for PS3, PSP and PS Vita
- **Gallery** - Image gallery with Lightbox2 preview
- **Emulation** - Emulator hub with AJAX-loaded data table
- **Under the Hood** - Console hardware deep dives
- **Contact** - Contact form with JS validation

## How to run

1. Clone the repository
2. Open `index.html` in a browser

Or use WebStorm's built-in server for full functionality (required for AJAX table to load).

## Technologies used

- HTML5
- CSS3 (Flexbox, responsive design, media queries)
- JavaScript (form validation, AJAX/fetch, burger menu)
- Lightbox2 for gallery
- Google Fonts (Raleway)
